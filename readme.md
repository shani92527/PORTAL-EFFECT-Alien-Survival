# 🚀 PORTAL EFFECT: Alien Survival

## 📌 Overview
PORTAL EFFECT: Alien Survival is a sci-fi action shooter game developed using Unity. The game places players in a hostile alien galaxy where they must survive against dangerous enemies, explore multiple planets, and use ancient portals to find their way back home.

The game combines fast-paced combat, exploration, and puzzle-solving elements to create an immersive survival experience.

---

## 🚀 Features
- 🛸 Intense alien combat with shooting mechanics  
- 🌌 Exploration of multiple planets and environments  
- 🌀 Portal-based progression system  
- 👾 Variety of enemies (drones, reptilian overlords, alien creatures)  
- 💎 Resource collection (diamonds and upgrades)  
- 🔫 Weapon upgrade system  
- 🎮 Smooth and responsive gameplay  
- 📱 Mobile-optimized performance  

---
